    Calculadora Interactiva

    v2.0

    Proyecto desarrollado con HTML, CSS y JavaScript puro. Demuestra buenas prácticas de arquitectura, accesibilidad y calidad de código.

    • Parser de expresiones — algoritmo Shunting-Yard (precedencia real)
    • Funciones científicas: sin, cos, tan, √, log, ln, x², n!
    • Memoria: MC, MR, M+, M−
    • Historial persistente con marcas de tiempo (localStorage)
    • Modo oscuro / claro guardado entre sesiones
    • Accesibilidad WCAG 2.1: ARIA, teclado, lector de pantalla
    • Retroalimentación sonora opcional (Web Audio API)
    • Diseño responsivo móvil / escritorio
